# AI 유튜브 요약 - 서비스 기획서

## 1. 서비스 개요

### 서비스 명
**TubeDigest AI** (튜브다이제스트 AI)

### 한 줄 소개
내 유튜브 구독 리스트를 AI가 자동 요약하고, 영어/일본어/중국어 영상까지 검색하여 번역 요약까지 제공하는 AI 영상 콘텐츠 큐레이션 서비스

### 핵심 가치 제안
- **구독 리스트 자동 요약**: 내가 구독한 채널들의 새 영상을 AI가 자동 요약하여 알림
- **다국어 검색 요약**: 영어, 일본어, 중국어 영상까지 AI가 검색하고 번역 요약
- **시간 절약**: 1시간 영상을 1분 요약으로, 핵심만 빠르게 파악
- **개인화 피드**: 내 관심사에 맞는 영상만 AI가 큐레이션

## 3. 기술 아키텍처

### AI/ML 스택
- **STT**: Whisper API
